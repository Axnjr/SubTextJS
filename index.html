Example

1. Add the below cdn at the bottom of the Html file.

"https://cdn.jsdelivr.net/gh/Axnjr/SubText.js@main/SubText.JS"


2. Assign class "SplitedText" to the parent element


It helps SubTextJS to identify which element has to be splited so it's neccesary.


3. As per your need, assign the below attributes.

Inline Attribute Description
splitType (Optional) Specifies the type of split of text either letters or words . Default type is letters.
subElementTag (Optional) Specifies the sub element tag name like "span" , "div", etc. Default value is undefined. While using gsap for animation prefer div tags and give " display : inline-block " property to sub element.
subElementClass (Optional) Specifies the class name to be assigned to the sub element created. Default class name is null
text (Compulsory) Text to be split goes in the text attribute , Instead of the element.

4. Use Html space entity to add spaces between words while using word split type.
